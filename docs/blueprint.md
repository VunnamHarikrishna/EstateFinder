# **App Name**: Pavan EstateFinder

## Core Features:

- Property Listing & Filtering: User-facing display of available properties, with features for browsing, searching, and filtering by location, price (low to high), and property type (plot, apartment, land). Includes rotating banners for offers.
- Detailed Property View: Comprehensive property details page showcasing all images, price, address, contact information, availability status, and amenities.
- Site Visit Request Form: Allows users to submit a site visit booking request by providing their name, mobile number, email, preferred date, and transportation requirements.
- Admin Authentication: Secure login system for administrators to access and manage the backend of the application.
- Property Management (CRUD): Admin interface to create, read, update, and delete property listings, including managing property images, pricing, details, and availability.
- AI-Powered Property Description Tool: A tool for administrators to generate detailed and engaging property descriptions using AI, based on structured input of key property features and selling points.
- Site Visit Request Management: Admin view to track, review, and manage all incoming site visit requests, including options to mark as processed or assign to agents.

## Style Guidelines:

- Primary color: A warm, inviting terracotta (#D17547) to convey a sense of home and aspiration, making it suitable for headings and interactive elements.
- Background color: A very light, warm off-white (#F5F1F0) providing a clean, spacious, and modern canvas that is gentle on the eyes.
- Accent color: A vibrant and elegant rosy red (#E4677C) used sparingly for calls-to-action, highlights, and important notifications to draw user attention.
- Secondary accent color: A rich, deep purple (#6A0DAD) for exclusive elements or premium features to add a touch of luxury and sophistication.
- All text (headlines and body): 'Inter', a grotesque sans-serif font chosen for its modern, objective, and highly legible characteristics across various screen sizes, promoting a reliable and trustworthy feel.
- Utilize modern, clean, outline-style icons that are universally recognized for actions like search, filters, contact, and property details, ensuring intuitive navigation.
- Implement a spacious, grid-based layout for property listings to allow ample breathing room for images and brief details, enhancing visual appeal and readability. Use clear segregation between different sections of content on detail pages.
- Subtle and smooth transition animations for rotating banners and when filters are applied, providing a dynamic yet non-distracting user experience.
